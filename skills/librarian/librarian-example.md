# Librarian Example

## Input

Abbiamo creato le prime sette skill. Aggiorna la documentazione del portfolio senza aggiungere file inutili.

## Skill Used

librarian

## Output

Updated documents:

- `README.md`

Preserved decisions or facts:

- Le prime sette skill sono state aggiunte.
- Il portfolio usa ruoli separati invece di un prompt generico unico.
- Gli esempi, i case study, il changelog e la licenza fanno parte del proof layer del portfolio.

Omitted details and why:

- Il changelog esiste già, quindi non è stato creato un nuovo file: è stata aggiornata solo la sezione pertinente.
- Non e' stato creato un documento separato per ogni decisione perché sarebbe prematuro.

Remaining documentation gaps:

- Mancano un breve demo workflow pubblico.
- Mancano before/after examples.
- Resta da verificare se gli esempi coprono casi reali sufficientemente diversi.

## Why This Matters

Questo output è migliore di una risposta generica perché aggiorna solo il documento giusto, evita nuova documentazione superflua e registra anche cosa resta da fare.
